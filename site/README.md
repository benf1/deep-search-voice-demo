# Publish To GitHub Pages

1. Push this folder to a repo.
2. In GitHub, set Pages source to deploy from `/deep-search-voice-demo/site` (or branch root if copied).
3. Ensure these files are present:
   - `index.html`
   - `assets/original-fr.mp3`
   - `assets/english-mix.mp3`
   - `assets/vocals-isolated.mp3`
   - `assets/music-bed.mp3`
4. Open the Pages URL and test both players.
